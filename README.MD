# Potaynik

"Потайник" - место, в котором можно спрятать свои секреты.

## Зависимости

* Unix среда
* Go
* Docker/docker-compose

## Установка & Запуск
1. Выполнить `make docker-create-network`
2. Запустить: `make run`

## Dev console
